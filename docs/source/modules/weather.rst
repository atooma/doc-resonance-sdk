Weather
--------------------------

* **Identifier** - WEATHER
* **Description** - Provides utilities for weather forecast and for monitoring current weather conditions.
* **Initialization** - None
* **Destroy** - None
* **Dependencies** - :ref:`module-core`, :ref:`module-location`
