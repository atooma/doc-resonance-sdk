SMS
--------------------------

* **Identifier** - SMS
* **Description** - Allows to send and receive SMS.
* **Current Version** - 1
* **Initialization** - None
* **Destroy** - None
* **Dependencies** - :ref:`module-core`, :ref:`module-contacts`, :ref:`module-telephony`
