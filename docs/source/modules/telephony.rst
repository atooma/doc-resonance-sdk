.. _module-telephony:

Telephony
--------------------------

* **Identifier** - TELEPHONY
* **Description** - Allows to control basic device features.
* **Initialization** - Looks for availability of mobile phone hardware (GSM).
* **Destroy** - None
* **Dependencies** - :ref:`module-core`, :ref:`module-contacts`
